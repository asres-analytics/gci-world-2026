# 🌍 GCI World 2026 — AI & Data Science Learning Journey

Welcome to my **GCI World 2026 learning journey**.

This repository documents my progress and hands-on learning through  
**GCI World 2026 (Global Consumer Intelligence)**, an international **AI & Data Science**
program offered by the **Matsuo‑Iwasawa Laboratory, University of Tokyo**.

Throughout this program, I focus on building strong foundations in **Python,
data analysis, machine learning, and applied AI**, while maintaining consistency
and clear documentation of my learning progress.

---

## 🎯 Purpose

The purpose of this repository is to:

- stay consistent with weekly learning
- build strong fundamentals in AI & Data Science
- apply concepts through hands-on practice
- organize learning in a structured, research-style format
- document progress publicly for reflection and growth

This repository serves as both a **learning log** and a **portfolio**.

---

## 🎯 Goals

The main goals of this learning journey are to:

- build a strong understanding of Python and data science fundamentals
- develop practical machine learning skills
- improve problem-solving through hands-on notebooks and projects
- strengthen discipline through consistent documentation
- create a well-structured AI learning portfolio

---

## 🏁 Final Outcome

By the end of **GCI World 2026**, I aim to have:

- a solid foundation in AI and data science concepts
- hands-on experience with real datasets and ML workflows
- completed mini-projects and applied exercises
- improved learning consistency and technical confidence
- a documented learning journey that I can always revisit

---

## 📅 Learning Journey Structure

This repository is organized **week by week**, following the structure of the
GCI World program.

Each weekly folder contains:

- `README.md` — weekly overview and objectives
- `dataset/` — datasets used during the week
- `notebooks/` — Jupyter notebooks and exercises
- `notes/` — summarized notes and reflections
