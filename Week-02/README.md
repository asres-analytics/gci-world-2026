## 🎯 Week Overview

Week 01 focuses on building a strong foundation in **Python programming**.
Python is the core language used throughout GCI World for data analysis,
machine learning, and applied AI tasks.

This week emphasizes understanding syntax, logic, and fundamental programming
constructs needed for later weeks.

---

## 🎯 Learning Objectives

By the end of this week, I aim to:

- Understand Python syntax and indentation rules
- Work confidently with variables and data types
- Use conditional logic and loops
- Write and use simple Python functions
- Become comfortable running code in Jupyter notebooks

---

## 📘 Topics Covered

- Python syntax and code structure
- Variables and built-in data types
- Arithmetic, comparison, and logical operators
- Conditional statements (`if`, `elif`, `else`)
- Looping structures (`for`, `while`)
- Function definition and return values
- Introduction to basic data structures
